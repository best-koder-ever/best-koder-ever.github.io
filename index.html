Martin Chagallo | Repo overview

DatingApp is a full-stack, AI-assisted engineering project built as a distributed dating platform across multiple repositories. It combines .NET 8 microservices, real-time SignalR messaging, Keycloak-based authentication, Firebase phone and SMS verification, external LLM moderation paths, and a Flutter mobile/web client to demonstrate production-style system design and delivery.

Single navigation page for all repositories

System Overview

  • Architecture: Microservice platform with dedicated services for user profiles, matchmaking, swipes, messaging, photos, safety, and bot orchestration behind YARP.
  • Identity and Verification: Keycloak OIDC handles session identity while Firebase supports phone auth, OTP/SMS flows, and push notification channels.
  • AI and Safety: Messaging moderation and safety policies call external LLM providers (Groq, Gemini, Ollama fallback) via safety-service patterns.
  • Client and Runtime: Flutter app targets web and mobile with real-time SignalR updates, plus event-driven calls across services and per-service persistence stores.
