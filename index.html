Martin Chagallo | Repo overview

DatingApp is a full-stack, AI-assisted engineering project built as a distributed dating platform across multiple repositories. It combines .NET 8 microservices, real-time SignalR messaging, Keycloak-based authentication, and a Flutter mobile/web client to demonstrate production-style system design and delivery.

Single navigation page for all repositories

System Overview

  • Architecture: Microservice platform with dedicated services for user profiles, matchmaking, swipes, messaging, photos, safety, and orchestration through a gateway.
  • Real-Time + Domain Logic: SignalR messaging, swipe-to-match workflows, and candidate scoring to show event-driven and domain-heavy backend design.
  • AI and Safety: Moderation and safety pipeline patterns integrated with messaging and media flows.
  • Client + Delivery: Flutter app for mobile/web, multi-repo workflow, and testable service boundaries with CI-ready structure.
