DatingApp Platform Overview

DatingApp is a full-stack, AI-assisted engineering project built as a distributed platform across multiple repositories. It combines .NET 8 microservices, real-time SignalR messaging, Keycloak identity, Firebase verification and push channels, external LLM moderation paths, and a Flutter mobile/web client to demonstrate production-style architecture and delivery.
