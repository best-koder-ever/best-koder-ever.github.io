    participant App as Flutter App
    participant Y as YARP Gateway
    participant SW as swipe-service
    participant MM as MatchmakingService
    participant MSG as messaging-service
    participant SAF as safety-service
    participant LLM as External LLM

    participant FCM as Firebase/FCM

    App->>KC: OIDC login (PKCE)
    KC-->>App: Access and refresh tokens

    App->>Y: POST /api/swipes (Bearer token)
    Y->>SW: Forward swipe request
    SW->>MM: Evaluate candidate and match rules
    MM-->>SW: Match result (yes/no)
    SW-->>Y: Swipe processing result
    Y-->>App: Swipe API response

    alt Match created
      MM->>MSG: Create conversation bootstrap
      MSG->>FCM: Push notification trigger
      FCM-->>App: New match notification
    end

    App->>Y: Send chat message
    Y->>MSG: Forward message
    MSG->>SAF: Moderate content
    SAF->>LLM: Classification request
    LLM-->>SAF: Safety verdict
    SAF-->>MSG: Allow/flag result
    MSG-->>App: SignalR realtime update
